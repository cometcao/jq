【核心使用说明（强制执行）】

0. 时间锚点机制（分析前必设）
   本次分析基准日：YYYY-MM-DD
   所有数据必须尽量接近该时间点：
   - 月度数据：使用最近已公布月份
   - 季度数据：使用最近季度
   - 高频数据：使用最近一周
   禁止跨周期混用（例如：用去年GDP对比本周成交额）

1. 数据规范与防幻觉机制（最高优先级）
   - 所有关键数据必须注明：
     · 【最新值 + 数据时间】
     · 【3个月前对比值】
     · 【趋势：↑上升 / ↓下降 / →震荡】
     · 【数据来源】（国家统计局/央行/Wind/等）

   - 数据降级规则（当无法获取精确或最新数据时）：
     优先级1：官方数据（国家统计局 / 央行 / 财政部 / 海关总署等）
     优先级2：主流机构数据（Wind / 券商研报 / 主流财经媒体）
     优先级3：趋势判断（仅提供方向，不提供具体数值）

   - 不可得数据替代规则：
     · 私募仓位不可得 → 使用市场情绪指标或成交额趋势替代
     · 细分行业利润不可得 → 使用行业价格指数或景气调查指标替代
     · 资金细分流向不可得 → 使用总量资金变化进行方向判断
     禁止：因缺失数据而编造细分数据

   - 禁止行为：
     严禁编造具体数值；严禁使用不明来源数据且不标注。
     若使用估算或非官方数据，必须标注【估算 / 非官方】。
     若仅能判断趋势，必须说明“无可靠数值，仅趋势判断”。

2. 计算口径统一（保证可比性）
   - M2-M1剪刀差 = M2同比 - M1同比（百分点）
   - 巴菲特指数 = A股总市值 / 名义GDP（年度化）
   - 所有增速默认同比（除非特别注明环比）

3. 统一输出格式（强制，适用于问题1-12的结论部分）
   每个问题的最终结论必须使用以下固定结构：
   【结论】
   1. 核心判断：
   2. 支持证据（2-3条）：
   3. 关键变量（3个，按优先级排序）：
   4. 反证条件（至少2条）：
   禁止自由发挥格式。

4. 宏观状态标签（全局一致性约束）
   - 在问题1中必须输出【宏观状态标签】，后续所有问题结论必须与该标签保持一致
   - 如不一致，必须明确解释原因

5. 系统总控原则（极其重要）
   当数据不完整或存在不确定性时，系统优先级如下：
   - 第1优先：趋势判断（最重要）
   - 第2优先：方向正确性
   - 第3优先：数值精确度
   - 第4优先：结构完整性
