# KBarChan缠论分析插件安装说明

## 插件概述
基于kBar_Chan.py核心算法改编的通达信缠论分析插件，提供笔和线段分析功能。

## 文件说明
1. **TDXChanPlugin32.dll** - 主插件文件（32位版本）
2. **KBarChan_Formula.txt** - 完整缠论指标公式内容
3. **KBarChan_Simple_Formula.txt** - 简化版指标公式内容
4. **通达信插件安装说明.md** - 本文件

## 安装步骤

### 1. 复制DLL文件
将 `TDXChanPlugin32.dll` 复制到通达信的DLL目录：
- 通达信安装目录\T0002\dlls\
- 如果`dlls`目录不存在，请手动创建

### 2. 在通达信中绑定DLL函数
1. 打开通达信软件
2. 按`Ctrl+F`打开公式管理器
3. 选择"DLL函数"类别
4. 点击"新建"按钮

**注意**：通达信会将第一个绑定的DLL函数自动命名为`TDXDLL1`，因此公式中使用`TDXDLL1`而不是`Plugin2`。

### 3. 创建指标公式
1. 在公式管理器中，选择"技术指标公式"
2. 点击"新建"按钮  
3. 输入公式名称：`KBarChan缠论分析`
4. 将以下内容之一粘贴到公式编辑区：

**完整版本**：
```
笔端点信号:TDXDLL1(1,H,L,0);
线段端点信号:TDXDLL1(2,H,L,0);
DRAWTEXT(笔端点信号=1,H*1.01,'↑'),COLORYELLOW;
DRAWTEXT(笔端点信号=-1,L*0.99,'↓'),COLORYELLOW;
DRAWTEXT(线段端点信号=1,H*1.02,'段顶'),COLORFF8000;
DRAWTEXT(线段端点信号=-1,L*0.98,'段底'),COLORFF8000;
```

**简化版本**：
```
笔端点:TDXDLL1(1,H,L,0),NODRAW;
线段端点:TDXDLL1(2,H,L,0),NODRAW;
DRAWTEXT(笔端点=1,H*1.01,'↑'),COLORYELLOW;
DRAWTEXT(笔端点=-1,L*0.99,'↓'),COLORYELLOW;
```

5. 点击"确定"保存

**注意**：公式内容已保存在`KBarChan_Formula.txt`和`KBarChan_Simple_Formula.txt`文件中，可以直接复制使用。

### 4. 使用指标
1. 在K线图中，点击右键选择"主图指标"
2. 选择"KBarChan缠论分析"
3. 或直接在键盘输入`KBarChan`然后回车

## 功能介绍

### 笔分析（黄色虚线）
- **↑标记**：笔的顶分型端点
- **↓标记**：笔的底分型端点
- **黄色虚线**：笔的连接线

### 线段分析（橙色实线）
- **段顶标记**：线段的顶分型端点
- **段底标记**：线段的底分型端点
- **橙色实线**：线段的连接线

## 技术说明

### DLL导出函数
插件包含以下标准通达信函数：
1. `Plugin2` - 核心分析函数
2. `CalcN` - 标准计算函数
3. `TDXPlugin_GetInfo` - 插件信息
4. `TDXPlugin_Init` - 初始化函数
5. `TDXPlugin_Calculate` - 计算函数
6. `TDXPlugin_Draw` - 绘图函数

### 算法特点
- 基于kBar_Chan.py完整缠论算法
- 支持任何周期的K线数据
- 实时更新分析结果
- 只包含笔和线段分析，无中枢功能（简化版）

## 常见问题

### 1. DLL绑定失败
- 确保DLL文件已复制到正确的目录
- 确认通达信版本（32位/64位）
- 如果通达信是32位版本，需要使用Visual Studio编译的32位DLL

### 2. 指标不显示
- 检查公式是否正确导入
- 确认`Plugin2`函数已正确绑定
- 尝试重新启动通达信软件

### 3. 分析结果不准确
- 确认K线数据完整
- 不同周期的分析结果会有差异
- 算法基于标准的缠论定义

## 注意事项
1. 插件基于kBar_Chan.py算法，分析结果可能与手动画图有细微差异
2. 只提供笔和线段分析，中枢功能已去除
3. 建议在较大时间周期（日线、60分钟）使用效果更佳
4. 首次使用可能需要加载历史数据才能正确分析
